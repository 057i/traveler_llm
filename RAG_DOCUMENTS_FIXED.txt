## API端点

### GET /api/documents/statistics
返回统计数据，包括：
  - 文档总数
  - 省份、城市、景点数量
  - 文本RAG和图RAG数据量

### GET /api/documents/progress/{task_id}
SSE流式返回处理进度：
  - event: start
  - event: progress (step, status, message, progress)
  - event: step_complete
  - event: complete
  - event: error

### POST /api/documents/upload
上传PDF文档并启动后台处理

### GET /api/documents/list
获取文档列表（分页）

## 数据来源

### ChromaDB (Vector Database)
  - 存储景点向量
  - metadata包含：name, description, location, province, city, tags
  - 统计：总景点数、省份数、城市数

### Neo4j (Knowledge Graph)
  - 存储景点节点和关系
  - 查询：MATCH (n:Destination) RETURN count(n)
  - 统计：图谱节点数

### Redis
  - 存储文档任务信息
  - 存储处理进度
  - 列表：document:tasks
