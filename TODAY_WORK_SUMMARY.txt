### 二、文档列表字段显示 ✅
**问题：** 省份、城市、提取景点数、页数、上传时间都显示为空或0

**解决方案：**
1. **统计API修复** - documents.py
   ```python
   # 从Destination节点字段提取省份城市（去重）
   MATCH (n:Destination)
   WHERE n.province IS NOT NULL AND n.province <> ''
   RETURN count(DISTINCT n.province) as count
   ```

2. **文档列表API修复** - documents.py
   - 移除Neo4j查询逻辑
   - 直接从Redis返回数据
   - 设置默认值

3. **Neo4j节点创建修复** - neo4j_client.py
   ```python
   SET d.province = $province,
       d.city = $city,
       d.filename = $filename
   ```

4. **PDF解析节点** - node_pdf_parse.py
   - 使用PyPDF2获取页数
   - 保存到state['pages_count']

5. **Finalize节点** - node_finalize.py
   - 统计destinations_count
   - 提取第一个景点的省份和城市
   - 保存pages_count到Redis
   - 更新Redis任务信息

6. **文档任务服务** - document_task.py
   - 添加`upload_time`字段

**效果：**
- ✅ 省份：江西省
- ✅ 城市：上饶市
- ✅ 提取景点数：35
- ⚠️ 页数：需要重新上传测试
- ✅ 上传时间：已添加字段
