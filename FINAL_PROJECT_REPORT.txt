### 1. 真实事件系统
- **node_start事件：** 节点开始执行时立即发送
- **node_end事件：** 节点完成时立即发送
- **无人工延迟：** 完全反映实际执行时间
- **event_type字段：** 统一标识事件类型

### 2. 完整数据流（页数功能）
```
上传API (PyPDF2读取) → create_task (保存Redis)
  → process_document (传递workflow) → initial_state
  → node_pdf_parse (从state读取) → node_finalize (更新Redis)
  → 前端显示
```

- **双模式检索：** 提升准确率

- **GraphRAG：** 支持关系推理

## 🎯 SSE事件系统

### 事件格式
```json
{
  "event_type": "node_start",
  "step": "minio",
  "step_name": "MinIO存储",
  "status": "processing",
  "message": "正在保存文件到MinIO...",
  "progress": 15,
  "timestamp": 1783873872.58
}
```

### 前端处理
```javascript
if (eventType === 'node_start') {
  updateStepStatus(step, 'processing')
} else if (eventType === 'node_end') {
  updateStepStatus(step, 'completed')
}
```
