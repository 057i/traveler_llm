### 步骤2: 检查前端代码

**需要检查的文件：**
- `frontend/src/views/DocumentManager.vue`

**需要确认：**

## 💡 可能的前端问题

### 问题1: 字段名不匹配
```javascript
// 错误示例
const textRAG = response.text_rag  // 下划线，错误
const graphRAG = response.graph_rag

// 正确应该是
const textRAG = response.textRAG  // 驼峰命名
const graphRAG = response.graphRAG
```

### 问题2: API未调用
```javascript
// 可能没有调用statistics API
// 或者调用的是其他API
```

### 问题3: 响应处理错误
```javascript
// 可能使用了错误的字段
const textRAG = response.data.textRAG  // 多了一层.data?
```
