## 📋 诊断步骤

### 步骤1: 检查后端日志

**查找以下关键字：**

1. **实体提取日志**
```
[Extract] Total extracted: X destinations
```
- 如果X > 0：提取成功
- 如果X = 0：提取失败，可能是Qwen API问题

2. **向量化日志**
```
[Vector] Generating embeddings for X destinations
[Vector] Hybrid vectorization complete: X destinations
```
- 如果有这些日志：向量化执行了
- 如果没有：向量化节点没执行

3. **图谱构建日志**
```
[Vector] GraphRAG completed with X entities
```
- 如果有：Neo4j插入执行了
- 如果没有：图谱节点没执行

---

### 步骤2: 检查Milvus数据

**在backend目录执行：**
```bash
cd backend

python -c "
import sys

from app.core.milvus_hybrid_client import get_milvus_hybrid_client
```

### 步骤3: 检查Neo4j数据

**在backend目录执行：**
```bash
python -c "
import sys
sys.path.insert(0, '.')

from app.core.neo4j_client import get_neo4j_client

neo4j = get_neo4j_client()

# 检查Destination节点
result = neo4j.query('MATCH (n:Destination) RETURN count(n) as count')
dest_count = result[0]['count'] if result else 0
print(f'Neo4j Destination节点数: {dest_count}')

# 检查Province节点
result = neo4j.query('MATCH (n:Province) RETURN count(n) as count')
province_count = result[0]['count'] if result else 0
print(f'Neo4j Province节点数: {province_count}')

# 检查City节点
result = neo4j.query('MATCH (n:City) RETURN count(n) as count')
city_count = result[0]['count'] if result else 0
print(f'Neo4j City节点数: {city_count}')

if dest_count == 0:
    print('❌ Neo4j是空的！图谱构建失败')
else:
    print(f'✓ Neo4j有 {dest_count} 个景点节点')
"
```
