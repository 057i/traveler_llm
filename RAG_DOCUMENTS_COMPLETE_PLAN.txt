### 任务3: 完善SSE推送
**文件：** `backend/app/workflows/document_processing/workflow_service.py`

**在每个节点执行时推送两次：**

```python
async for event in self.workflow.astream(initial_state):
    for node_name, node_state in event.items():
        step_id = self.STEP_MAPPING.get(node_name)

        if step_id and step_id not in completed_steps:
            # 1. 节点开始 - 推送 progress 事件
            await self._send_step_event(
                task_id=task_id,
                step_id=step_id,
                status="processing",
                message=f"正在{step_name}...",
                progress=self.STEP_PROGRESS[step_id]
            )

            # 等待节点执行...

            # 2. 节点完成 - 推送 step_complete 事件
            await self._send_step_event(
                task_id=task_id,
                step_id=step_id,
                status="completed",
                message=f"{step_name}完成",
                progress=self.STEP_PROGRESS[step_id]
            )
```
