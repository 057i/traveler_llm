```python
# 对每个重写查询都做检索
for query in rewritten_queries:
    results = milvus.search(query, top_k=10)
    all_results.extend(results[:5])  # 每个查询取Top5

# 去重合并（按title，保留最高分）
unique_results = deduplicate(all_results)
```

### 3. 置信度判断优化
```python
# 从平均分改为最高分
max_score = max(scores)

# 阈值从0.6降到0.5（更严格）
if max_score < 0.5:

```

### 4. 统一Rerank
```python
# 合并所有来源
all_results = (
    milvus_results +      # 向量检索

)

# 统一重排序
final_top5 = rerank(all_results, top_k=5)
```
