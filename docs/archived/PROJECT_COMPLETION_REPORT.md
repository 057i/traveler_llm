## ⚙️ 系统配置

### 后端配置
```python
# Milvus混合检索权重
MILVUS_DENSE_WEIGHT: 0.8   # 稠密向量（语义相似度）
MILVUS_SPARSE_WEIGHT: 0.2  # 稀疏向量（关键词匹配）

# RRF融合参数
RRF_K: 40  # 影响融合后的分数

# RAG配置
RAG_TOP_K: 5
RAG_SIMILARITY_THRESHOLD: 0.7
```

// 路由配置 (动态菜单)
