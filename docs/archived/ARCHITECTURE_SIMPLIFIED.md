## 📊 最终架构

```
用户问题 + 历史会话
    ↓
[1. 查询理解]
  → 指代词消解
  → 实体提取: ["故宫"]
  → 意图识别: nearby_food
  → 生成1个重写查询 ✅
    ↓
[2. 单查询检索] ✅
  重写查询 → Milvus → Top10
    ↓
[3. RRF融合]
  → 只对Milvus结果融合
  → Neo4j不参与
    ↓
[4. 置信度检查]
  max_score < 0.5?
    ├─ 是 → Tavily补充
    └─ 否 → 跳过
    ↓
[5. Neo4j附近推荐]
  if need_nearby = True:
    基于Top3推荐附近景点
    (单独返回)
    ↓
[6. 统一Rerank]
  Milvus + Tavily → Top5
  (Neo4j不参与)
    ↓
[7. LLM生成答案]
  主结果 + 附近推荐
```
