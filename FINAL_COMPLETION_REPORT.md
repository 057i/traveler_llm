## 🎯 最新修复详解

### **第26项：相关度分数修复**

**问题**：
```python
# 错误：使用RRF分数（很小，0.005左右）
"score": r.get('rrf_score')  # → 显示为 0.5%
```

**修复**：
```python
# 正确：使用Rerank分数（正常，0.9左右）
"score": r.get('rerank_score')  # → 显示为 95%
```
