## ⚠️ 常见问题

### 问题1: Docker未运行
**症状：** `Docker未运行`
**解决：** 打开Docker Desktop

### 问题2: 端口被占用
**症状：** `port is already allocated`
**解决：**
```bash
# 找到占用进程
netstat -ano | findstr :19530

# 结束进程或修改端口
```

### 问题3: 镜像下载慢
**症状：** 下载很久
**解决：** 配置Docker镜像加速器

## 📝 手动命令（备用）

如果脚本失败，可以手动执行：

```bash
# 创建并启动Milvus
docker run -d --name milvus-standalone -p 19530:19530 -p 9091:9091 milvusdb/milvus:latest

# 等待30秒
timeout /t 30

# 测试连接
cd backend
python -c "from pymilvus import connections; connections.connect('default', host='localhost', port=19530); print('✓ 连接成功')"
```
