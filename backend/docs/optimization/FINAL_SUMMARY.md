## 🎯 技术要点

### 为什么归一化能解决问题？

**问题根源**：
```python
# 原始加权平均
fused = 0.9 × 0.72 + 0.1 × 0.35 = 0.6505  ❌ 损失10%
```

**归一化方案**：
```python
# 步骤1: 归一化到 [0, 1]
dense_norm = (0.72 - 0.61) / (0.72 - 0.61) = 1.0
sparse_norm = (0.35 - 0.18) / (0.35 - 0.18) = 1.0

# 步骤2: 加权平均
fused_norm = 0.9 × 1.0 + 0.1 × 1.0 = 1.0

# 步骤3: 缩放回原始范围
final = 1.0 × 0.72 = 0.72  ✅ 保留100%
```

**关键优势**：
- 消除尺度差异（dense在0.6-0.7，sparse在0.2-0.4）
- 公平对待两个来源
- 保留相似度语义（分数仍有意义）
