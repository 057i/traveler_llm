## 🔍 工作流分析

### 你的 LangGraph 工作流

**文件**: `app/workflows/ai_recommend/graph_builder.py`

```python
问题重写 (query_rewriter)
  ↓
并行检索 (parallel_retrieval)  # Milvus + Neo4j
  ↓
RRF融合 (rrf_fusion)           # 第一次融合
  ↓
置信度检查 (confidence_check)
  ↓
Tavily搜索 (tavily_search)      # 低置信度时触发
  ↓
RRF融合2 (rrf_fusion_with_tavily)  # 融合Tavily结果
  ↓
Rerank精排 (rerank)             # 🎯 使用 bge-reranker-v2-m3
  ↓
LLM润色 (synthesizer)
  ↓
返回结果
```

**Rerank节点**（`node_rerank.py`）:
- 使用 `app.core.rerank_engine`
- 模型: `bge-reranker-v2-m3`
- 输入: RRF融合后的结果
- 输出: 精排后的Top 5
