## 日志级别建议

- **INFO**: 关键流程和结果
- **DEBUG**: 详细计算示例（默认不显示，需要时开启）
- **WARNING**: 汇总的过滤信息
- **SUCCESS**: 最终结果

可以通过设置日志级别控制详细程度：
```python
# 正常使用：只显示INFO及以上
logger.setLevel("INFO")

# 调试模式：显示所有DEBUG细节
logger.setLevel("DEBUG")
```
