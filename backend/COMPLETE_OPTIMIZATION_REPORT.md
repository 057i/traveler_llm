### 第三轮优化：修复RRF虚高问题
**时间**: 2026-07-12 下午
**目标**: 解决RRF分数虚高（1.0）问题

#### 优化7: RRF 分数缩放优化
- **修改**: `fusion_strategies.py` - RRFFusion.fuse()
- **问题**: RRF归一化导致最高分永远是1.0
- **方案**: 缩放到原始Dense分数范围
- **效果**: 分数从虚高的1.0变为真实的0.70+

**修改内容**:
```python
# 修改前
rrf_scores[doc_id] = rrf_scores[doc_id] / max_rrf  # → 1.0

# 修改后
normalized = (score - min_rrf) / (max_rrf - min_rrf)
final = normalized * (max_dense - min_dense) + min_dense  # → 0.70
```
