## 🔧 使用方式

### 线上API（推荐）

**优势：**
- ✅ 无需本地部署
- ✅ 云端高性能处理
- ✅ 自动更新维护
- ✅ 按量付费

**配置步骤：**

1. **获取API密钥**
   - 访问MinerU官网（请提供实际地址）
   - 注册账号
   - 获取API Key

2. **配置.env**
   ```bash
   cd backend
   cp .env.example .env
   # 编辑.env，填入实际的MINERU_API_KEY
   ```

3. **启动后端**
   ```bash
   python main.py
   ```

4. **测试上传**
   - 访问：http://localhost:5173/documents
   - 上传PDF文件
   - 观察实时进度和日志

## 📊 API调用流程

```
用户上传PDF
    ↓
后端保存文件
    ↓
读取settings.MINERU_API_URL和MINERU_API_KEY
    ↓
POST请求到MinerU API
  Headers: Authorization: Bearer {api_key}
  Body: multipart/form-data (file)
    ↓
等待响应（1-30秒）
    ↓
解析JSON获取markdown字段
    ↓
保存到state['parsed_text']
    ↓
继续后续处理（分块、提取、向量化）
```
