## 📊 SSE事件流

**上传后将看到：**
```javascript
1. start: 开始处理文档
2. step: 'minio', status: 'processing'      // MinIO存储中
3. step_complete: 'minio'                    // MinIO完成
4. step: 'parse', status: 'processing'       // PDF解析中
5. step_complete: 'parse'                    // PDF解析完成
6. step: 'chunk', status: 'processing'       // 文本分块中
7. step_complete: 'chunk'                    // 文本分块完成
8. step: 'extract', status: 'processing'     // 实体提取中
9. step_complete: 'extract'                  // 实体提取完成
10. step: 'vectorize_traditional'            // 传统向量化中
11. step_complete: 'vectorize_traditional'   // 传统向量化完成
12. step: 'vectorize_graph'                  // 图向量化中
```
