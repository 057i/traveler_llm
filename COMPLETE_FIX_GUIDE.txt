## ⚠️ 为什么之前有132个Neo4j节点但统计为0？

**原因：** 统计API查询的是：
```cypher
MATCH (n:Province) RETURN count(n)  // 查询Province标签
MATCH (n:City) RETURN count(n)      // 查询City标签
```

但之前创建的节点只有：
```cypher
MATCH (n:Destination) RETURN count(n)  // 132个Destination节点
```

**没有单独的Province和City节点！**

**解决方案：** 修改统计逻辑，从Destination节点中提取省份和城市：

```cypher
// 统计省份数
MATCH (n:Destination)
WHERE n.province IS NOT NULL AND n.province <> ''
RETURN count(DISTINCT n.province) as count

// 统计城市数
MATCH (n:Destination)
WHERE n.city IS NOT NULL AND n.city <> ''
RETURN count(DISTINCT n.city) as count
```
