# ✅ 最终配置 - 简化版置信度计算

## 🎯 当前方案

### **置信度计算（简化版）**

**3个维度，权重固定**：

1. **精确匹配（50%权重）**
   - Top1精确匹配：+0.5
   - Top3精确匹配：+0.4
   - 有精确匹配但靠后：+0.3
   - 无精确匹配：+0.1

2. **结果数量（30%权重）**
   - ≥10条：+0.3
   - ≥5条：+0.2
   - ≥3条：+0.15
   - <3条：+0.1

3. **来源多样性（20%权重）**
   - 3种来源（稠密+稀疏+精确）：+0.2
   - 2种来源：+0.15
   - 1种来源：+0.1

**总分范围**: 0.4 - 1.0

- `0.6` - 平衡（推荐）
