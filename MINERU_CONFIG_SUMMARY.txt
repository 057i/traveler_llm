## 🔧 使用MinerU线上API

### 优势

✅ **无需本地部署** - 不需要安装和维护MinerU服务
✅ **云端处理** - 高性能服务器处理PDF
✅ **自动更新** - 官方维护，始终使用最新版本
✅ **按量付费** - 只为使用的量付费
✅ **高可用** - 云服务保证稳定性

### API调用流程

```
用户上传PDF
    ↓
MinIO存储节点 (保存文件)
    ↓
PDF解析节点 (调用MinerU API)
    • 读取settings.MINERU_API_URL
    • 读取settings.MINERU_API_KEY
    • 构建POST请求
    • 上传PDF文件到MinerU云端
    • 等待处理（1-30秒）
    • 解析JSON响应获取markdown字段
    • 保存到state['parsed_text']
    ↓
文本分块节点
    ↓
实体提取节点
    ↓
并行向量化节点
    ↓
完成节点
```

### 请求示例

```python
# 请求
POST https://api.mineru.com/v1/parse
Headers:
  Authorization: Bearer {your_api_key}
  Content-Type: multipart/form-data
Body:
  file: [PDF文件二进制数据]

# 响应
{
  "markdown": "# 文档标题\n\n文档内容...",
  "pages": 10,
  "status": "success"
}
```

### 错误处理

代码已处理所有可能的错误：

| 错误类型 | HTTP状态 | 处理方式 |
|---------|---------|---------|
| API密钥无效 | 401 | 记录错误，提示检查密钥 |
| 配额不足 | 429 | 记录错误，提示配额用尽 |
| 文件格式错误 | 400 | 记录错误，提示文件格式 |
| 服务器错误 | 500 | 记录错误，建议稍后重试 |
| 连接超时 | Timeout | 记录错误，显示超时信息 |
| 网络错误 | Connection | 记录错误，检查网络连接 |
